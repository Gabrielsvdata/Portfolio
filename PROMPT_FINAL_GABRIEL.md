PROJETO 1: SISPAR (Sistema de Pedidos e Requisições)
─────────────────────────────────────────────────────

Nome: SISPAR - Sistema de Pedidos e Requisições
GitHub: https://github.com/Gabrielsvdata/Sispar-Backend
Live: https://projeto-sispar.vercel.app/
Tech Stack: Node.js, Express, PostgreSQL, React, TypeScript, Vercel

Descrição:
Sistema completo de gerenciamento de pedidos e requisições para empresas.
Integração com múltiplos módulos, autenticação robusta, dashboard em tempo real.

Problema:
Empresas precisam gerenciar centenas de pedidos/requisições manualmente.
Processo lento, propenso a erros, sem visibilidade em tempo real.

Solução:
Sistema integrado com API robusta em Node.js + Frontend React.
Automação de workflows, notificações em tempo real, relatórios detalhados.

Impacto:
✓ 5k+ requisições gerenciadas por mês
✓ Redução de 70% no tempo de processamento
✓ 99.5% uptime em produção
✓ 50+ usuários ativos

Color: primary (laranja)
